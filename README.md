# ༄ بوت رعد — Raad Bot

بوت تيليغرام عربي متكامل لإدارة المجموعات، مبني بـ Python + Pyrogram + Redis.

## ⚙️ التثبيت

### المتطلبات
- Python 3.10+
- Redis Server
- ffmpeg (للتحميل)

### الخطوات

```bash
# 1. انسخ المشروع
cd bot

# 2. شغّل سكريبت التهيئة
chmod +x setup.sh && ./setup.sh

# 3. عدّل ملف الإعدادات
nano .env

# 4. شغّل البوت
python3 main.py
```

## 🔐 مبادئ الأمان

1. **عزل المفاتيح:** كل مفتاح Redis يبدأ بـ `raad:{BOT_ID}:` — لا تعارض بين بوتات متعددة.
2. **تحقق الصلاحيات:** كل أمر يتحقق من رتبة المستخدم قبل التنفيذ.
3. **حماية الرتب:** لا يمكن منح رتبة أعلى من رتبتك.
4. **Threading:** كل عملية ثقيلة في thread منفصل.
5. **معالجة الأخطاء:** try/except على كل عملية مع رسالة واضحة.
